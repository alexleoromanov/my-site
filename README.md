# StrikeZone Neon Bowling Site

Welcome to the **StrikeZone** official website and staff dashboard. Experience the future of bowling with our state-of-the-art neon lanes and interactive terminal.

## Quick Start

To run the site locally on `localhost:3000`, follow these simple steps:

1.  **Prerequisites**: Ensure you have [Node.js](https://nodejs.org/) installed on your machine.
2.  **Installation**: No external dependencies (like Express) are required, as the server uses native Node.js modules.
3.  **Start the Server**:
    ```bash
    npm start
    ```
4.  **Access the Site**:
    - **Public Site**: [http://localhost:3000](http://localhost:3000)
    - **Staff Dashboard**: [http://localhost:3000/dashboard.html](http://localhost:3000/dashboard.html)

## Features

-   **Neon Aesthetic**: Full dark mode with vibrant neon-styled UI components.
-   **Dynamic Dashboard**: Real-time lane management, arrivals tracking, and interactive timetables.
-   **No-Dependency Server**: A lightweight, robust Node.js server to serve strictly what's needed.
-   **Responsive Design**: Mobile-friendly layouts for both the public and staff interfaces.

## Project Structure

-   `index.html`: The main public landing page.
-   `dashboard.html`: The staff control panel.
-   `server.js`: The backend server handling static file delivery.
-   `script.js`: All frontend interactivity and dashboard state management.
-   `style.css`: Unified styling system including glassmorphism and neon animations.
-   `package.json`: Project metadata and run scripts.
